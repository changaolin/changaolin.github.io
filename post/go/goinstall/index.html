brew 安装

`brew install go`

安装地址为 `/usr/local/go`

环境配置

# GO
export PATH=$HOME/go/bin:$PATH
export GOROOT=/usr/local/go
export GOPATH=$HOME/go

测试

  • mkdir -p $HOME/go/src/hello
  • hello.go
package main

import "fmt"

func main() {
    fmt.Printf("hello, world\n")
}
- `go build` 在该hello.go文件目录下运行 go build, 这个命令会创建一个命名为 hello 的可执行的文件
- `go install hello` 安装到工作空间 ($HOME/go) 的bin目录中-全局调用
- `go clean hello` 撤销安装
