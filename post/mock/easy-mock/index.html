安装

  • easy-mock-github

  • nvm :node版本要求为v8.9.1

  • docker安装mongo

    docker pull mongo
docker run -itd --name mongo -p 27017:27017 mongo

  • docker安装redis

    docker pull redis
docker run -itd --name redis -p 6379:6379 redis

  • docker 安装 swagger.edit

docker pull swaggerapi/swagger-editor
docker run -itd --name s-editot -p 8080:8080 swaggerapi/swagger-editor
启动后->file->download-> 得到 swagger.json

swagger.json.png

  • docker 安装swagger-ui[可选]
docker pull swaggerapi/swagger-ui
docker run -itd --name s-ui -p 80:8080 -e SWAGGER_JSON=swagger.json swaggerapi/swagger-ui

配置服务

easy-mock

  • 修改easy-mock/config/default.json
    • mongo地址配置
    • redis地址配置
npm install
npm run dev

启动端口 7300

建立成功 可以通过 http://localhost:7300/docs 查阅文档
